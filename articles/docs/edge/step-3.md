# Step 3 — Add Search Engine

Add EchoSearch to Edge.

## Instructions
- Click **Search engines**  
- Press **Add search engine**

### Input Details

Fill in the fields exactly as shown below.

| Field | Value |
|------|------|
| Search engine | EchoSearch |
| Shortcut | echo |
| URL with %s in place of query | ```https://echo-search.github.io/#gsc.tab=0&gsc.q=h&gsc.sort=``` |

> ⚠️ Make sure `%s` stays in the URL — this is what Edge uses to insert your search query.

## Result
You can now register a new search provider.
